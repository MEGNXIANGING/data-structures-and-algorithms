### 单例模式
    单例模式是生成对象的最佳是实践，一个类只有唯一的对象，生成对象的方法也在这个类中，直接调用方法来生成类，不需要自己new
    懒汉式： class Singe{
            pravte static Singe INSTANCE=new Singe();
            pravte Singe(){
            }
            public Singe getInstance(){
                return INSTANCE;
            }
    }
    缺点：无法进行懒加载，对象在类加载的时候就会生成
